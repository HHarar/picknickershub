Our Story

About Picknickers Hub

Built by friends, for friends.

What is Picknickers Hub?

Picknickers Hub is a free, open-source online gaming hub built so a group of friends can easily gather and play games together — no downloads, no accounts required, no subscription fees. Just pick a game and share the link.

Why did we build it?

We got tired of bouncing between a dozen different platforms for different games, each requiring a separate sign-up or a paid tier to play with more than two people. We wanted one place that worked for our group, so we built it ourselves.

Is it free?

Yes — completely free, always. The project is open source and hosted on GitHub Pages. There are no ads, no premium tiers, and no data harvesting.

Can I contribute?

Absolutely! Head over to the GitHub repository and open a pull request. Whether you want to build a new game, fix a bug, or improve the design — all contributions are welcome.

Contact

Open an issue on GitHub or reach out via the repository discussions tab. We'd love to hear from you.
